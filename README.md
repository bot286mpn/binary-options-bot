# Binary Options Trading Bot

Данный проект реализует автоматического бота для бинарных опционов на Python с анализом свечных паттернов и автоматизацией действий через Selenium и PyQt5.

## Основные возможности:
- Анализ паттернов свечей за несколько секунд до закрытия свечи
- Автоматизация открытия сделок через Selenium или PyAutoGUI
- Графический интерфейс для выбора окна и области захвата
- Логи и сигналы для торговли

## Установка:

1. Установите зависимости:
```
pip install -r requirements.txt
```

2. Запустите приложение:
```
python trading_bot.py
```

## Настройки
- Выберите область захвата экрана
- Укажите время до закрытия свечи для анализа
- Включите автоматическую торговлю (если требуется)
